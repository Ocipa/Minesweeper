# Minesweeper


## Build Steps
### Wally & Rojo
1. git clone https://github.com/Ocipa/Minesweeper.git
2. cd Minesweeper
3. wally install
4. rojo build --output ./Minesweeper.rbxl
5. Minesweeper.rbxl

### Releases
download the Minesweeper.rbxl in releases.
